LinPlayer · 桌面端 UI 重构 · 草稿评审 v1

桌面端页面草稿图

参考移动端的功能，但重做交互。桌面不是放大的手机——常驻侧栏取代浮动导航，右键菜单取代底部弹窗，键盘与悬停取代手势。下面是首批 4 个核心页面的布局草稿，橙色 n 标注每处「移动端做法 → 桌面做法」的差异，供你逐页核对。

方向 影院沉浸式 · Plex/网飞感 骨架 常驻左侧栏 · 操作内嵌内容区 首批 核心播放流 4 页

草稿=布局与交互，非最终视觉；灰色斜纹块代表图片/海报位。窗口宽度自适应，草稿按典型宽屏比例绘制。

贯穿外壳

左侧栏 · 服务器切换器就在栏内

移动端把「服务器列表」当作首页/退出锚点，靠浮动药丸在大区间跳转。桌面端反过来：服务器切换器常驻侧栏顶部，导航常驻侧栏中部，无需离开当前内容。侧栏可折叠为窄图标条（顶栏汉堡切换）。
